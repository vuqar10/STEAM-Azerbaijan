# Süni İntellekt və Robototexnika Dərs Proqramı Kodları

Bu, özəl olaraq hazırlanmış, 10 dərslik bir Süni İntellekt və Kompüter Görməsi əsaslı Robototexnika kursunun praktiki kod bazasıdır.

Layihənin məqsədi tələbələri əsas OpenCV funksiyalarından başlayaraq, mütənasib (Proportional) tənzimləməyə əsaslanan tam avtonom obyekt izləyən robot yaratmağa aparmaqdır.

## Dərslərin İcmalı

| Dərs № | Mövzu Adı | Yerləşdiyi Qovluq |
| :---: | :--- | :--- |
| 2 | OpenCV ilə Görüntü Emalına Giriş | Python Kodları |
| 3 | Rəngləri Tanıyaq (HSV Maska) | Python Kodları |
| 4 | Kompüterin Rəngə Görə Reaksiyası | Python Kodları |
| 5 | Obyektin Mərkəzini Tapmaq (Centroid) | Python Kodları |
| 6 | Serial Əlaqə: Beyin və Bədəni Birləşdirmə | Python Kodları & Arduino Kodları |
| 7 | Robotun Ayaqları: Motorlar və L298N | Arduino Kodları |
| 8 | Məsafədən İdarəetmə (WASD) | Python Kodları & Arduino Kodları |
| 9 | Avtonom Hərəkət: Kəskin Hədəfləmə | Python Kodları & Arduino Kodları |
| 10 | Həssas Sükan: Proportional Control | Python Kodları & Arduino Kodları |

## Tələblər

* **Python:** 3.x
* **Arduino IDE**
* **Kitabxanalar:** Lazım olan Python kitabxanaları `requirements.txt` faylında qeyd olunub.
